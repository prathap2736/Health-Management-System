# Health Management System (Premium Edition)

A fully responsive, modern, and premium-level Health Management System frontend built with React, Tailwind CSS, and Framer Motion. This project simulates a real-world SaaS dashboard product for hospitals and clinics.

## 🚀 Key Features

*   **Premium UI/UX:** Soft gradients, glassmorphism, micro-interactions, and smooth page transitions.
*   **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop.
*   **Dark Mode:** Built-in dark theme toggle with persistent state.
*   **Role-Based Dashboards:**
    *   **Patient Dashboard:** Appointment tracking, vitals visualization, and prescription history.
    *   **Doctor Dashboard:** Real-time analytics, patient charts, and appointment management.
    *   **Admin Dashboard:** System health, user management, and security logs.
*   **Advanced Components:**
    *   Animated Charts (Chart.js)
    *   Interactive Appointment Booking
    *   Digital Prescription Viewer
    *   Billing & Invoicing System
*   **Performance:** Optimized build with Vite.

## 🛠 Tech Stack

*   **Frontend:** React 19, Vite
*   **Styling:** Tailwind CSS 3.4
*   **Animations:** Framer Motion
*   **Icons:** Lucide React
*   **Charts:** Chart.js, React-Chartjs-2
*   **Notifications:** React Hot Toast
*   **Routing:** React Router DOM v7

## 📦 Installation

1.  Clone the repository or download the source.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open `http://localhost:5173` in your browser.

## 📂 Project Structure

*   `src/components`: Reusable UI components (Navbar, Footer, etc.)
*   `src/pages`: Full page views (Home, Login, Dashboards, etc.)
*   `src/index.css`: Tailwind directives and custom utility classes.
*   `tailwind.config.js`: Custom theme configuration.

## 🎨 Theme Customization

To modify the color scheme, edit `tailwind.config.js`. The primary brand colors are defined under `theme.extend.colors`.
