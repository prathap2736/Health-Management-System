🏥 Health Management System

A Health Management System is a modern web application designed to simplify healthcare management by digitally managing patient records, appointments, and healthcare data. This project was developed as a Final Year Project for M.Sc Computer Science.

🎯 Project Objective

The main objective of this project is to develop a digital platform that helps healthcare organizations efficiently manage patient information, appointments, and administrative operations in a secure and user-friendly environment.

✨ Key Features

🔐 Secure Login & Registration System

📊 Interactive Dashboard

👤 User Profile Management

📅 Appointment Management

📋 Patient Record Management

📱 Fully Responsive Design

🎨 Modern UI with Animations

✔ Form Validation

🔎 Search and Filter System
